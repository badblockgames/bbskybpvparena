Arène PVP SkyBlock

Le Jeu :
	Les joueurs du skyblock peuvent parier des items(config) pour combatre dans l'arene, s'il gagnent le combat, ils remporte leur mise.
Exemple : un joueur parie 45Blocks d'émeraude, un autre joueur accepte le parie (il parie donc lui aussi 45Blocks d'émeraude), le gagnat du combat remporte 90Blocks d'émeraude.

Les commandes :
  prefix: /arene

	/arene help/h/? : affiche la liste des commandes (commandes admin affichées uniquement pour les perm admins)
	/arene sign : Vous tp à la position du panneau (config)
	/arene sign [playerName] : (perm admin ou console) tp le joueur choisi devant le panneau
	/arene cancel : le joueur qui a lancer le parie peut l'annuler avant que quelqu'un accepte le combat
	/arene clearConfig : (uniquement via console) réinitialise la config par default
